# 递归式

一个递归式就是一个等式或不等式，
它通过更小的输入上的函数值来描述一个函数。

## 一、代入法

1. 猜测解的形式。  

2. 用数学归纳法求出解中的常数，并证明解是正确的。  

### 例子

求解: T(n) = T(n-1) + n  

    假设其解为O(n^2)，即 T(n) = O(n^2)
    代入法要求: 证明选择常数 c>0 ，可得 T(n) ≤ c*n^2
    假设对于 m<n ，总有 T(m) ≤ c*m^2
    即 T(n-1) ≤ c*(n-1)^2
    那么
    T(n) ≤ c*(n-1)^2 + n
         ≤ c*n^2 - c*2n + c + n
         ≤ c*n^2 - (c*2n - c -n)
         ≤ c*n^2 - ((n-1)c + (c-1)n)
    由于 n>1 && c>0
    故 T(n) ≤ c*n^2
    得 T(n) = O(n^2)
